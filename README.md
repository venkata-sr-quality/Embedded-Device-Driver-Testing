# Embedded & Device Driver Testing – Senior QA Portfolio

This repository showcases how I approach testing of
embedded systems and device drivers from a **Senior Test Engineer perspective**.

This is not a client project and contains no proprietary information.

## What This Repository Focuses On
- Embedded device driver validation
- Hardware–software interaction testing
- Boundary and failure condition handling
- Firmware lifecycle testing
- Stability, safety, and recovery behavior

## Systems Covered (Generic)
- Embedded controllers & drivers
- Hardware sensors and actuators
- Communication interfaces
- Firmware-based devices

The goal of this repository is to demonstrate
**how embedded quality is engineered**, not just verified.
